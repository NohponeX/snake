snake.js

Snake - Select images

Images can be url to image files for example :
https://pbs.twimg.com/profile_images/3702798811/e6f198f1600a04605ffd74e9c4b601de_normal.jpeg
or image encoded as base64 strings for example :
data:image/jpg;base64,/9j/4QAYRXhpZgAASUkqAAgAAAAAAAAAAAAAAP/...
Optimal image size : 24x24 pixel Leave empty to use default values
